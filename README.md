# Korean Saju MVP

한국식 사주를 기반으로 한 영문 개인 전략 리포트 서비스.

## 프로젝트 구조

- `data-collection/` — 천문연 API에서 만세력 데이터 수집
- `saju-engine/` — 사주 4기둥 계산 로직
- `ai-interpretation/` — Claude API 기반 해석 생성
- `pdf-generator/` — 영문 PDF 생성
- `web-app/` — Next.js 풀스택 앱

## 8주 로드맵

- W1: 만세력 DB 구축 (← 진행 중)
- W2: Claude 해석 프롬프트 설계
- W3: PDF 템플릿 디자인
- W4: 결제 + 자동화 (Lemon Squeezy + Resend)
- W5: 설문 + 대시보드
- W6: 월간 구독 자동 발송
- W7: 랜딩페이지 + Founding Member
- W8: 베타 테스트 + 출시
